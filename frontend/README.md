# Event Management Frontend

React frontend for event management application built with Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

The app will be available at `http://localhost:3000`

## Features

- View all events
- View upcoming events
- Create new events
- Responsive design

## Environment

The app proxies API calls to `http://localhost:5000` via Vite dev server.
Make sure the backend is running on port 5000.

## Build

```bash
npm run build
```

Output will be in `dist/` directory.
